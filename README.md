# Recipe_App_React (React + TypeScript)

The project implements a responsive web application using React for the frontend and TypeScript for type-safe development. A responsive web application with user registration and authentication (including persistent login), allowing users to search and view detailed recipes, as well as manage and update their personal profile information.

# Program execution instructions

1. Install the Node.js.
2. Open the project using Visual Studio Code.
3. Open a terminal in the root directory of the project.
4. Install the dependencies: npm install
5. Start the development server: npm run dev
6. In the browser, the application is usually available at:
   http://localhost:3000 or http://localhost:5173
